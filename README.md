# Lunchly

![Screenshot of Lunchly](http://i.imgur.com/GXtLcdw.png)

Lunchly is a Ruby on Rails app that helps your team figure out where to go for lunch.

## Development

  1. Configure your database by using `config/database.yml.example`
  2. then run:

    $ bundle install
    $ rake db:create
    $ rake db:migrate
    $ rake db:seed
    $ rails s
